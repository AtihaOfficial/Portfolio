##  Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Interactivity and form handling  
- **Google Apps Script** – Backend for contact form  
- **Google Sheets** – Stores contact messages  

---

##  Features

- Responsive navigation bar with dropdown menu
- Hero section with introduction
- About section with background image
- BPortfolio section with project showcase and GitHub links
- Blog section with external blog pages
- Contact form with:
  - Google Apps Script integration
  - Data stored in Google Sheets
  - Success & error messages
- Social media links
- Clean UI and visually appdealing design
